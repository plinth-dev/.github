# Code of Conduct

The Plinth community adopts the [**Contributor Covenant, version 2.1**](https://www.contributor-covenant.org/version/2/1/code_of_conduct/) as its code of conduct. The full text is available at:

https://www.contributor-covenant.org/version/2/1/code_of_conduct/

The Contributor Covenant is the de facto standard for open-source community conduct. By participating in any Plinth space — issues, discussions, pull requests, chat, in-person events — you agree to abide by it.

## Reporting

Report violations to **`conduct@plinth.run`**. Reports are read by the maintainers; the reporter is not identified to the subject of the report without consent.

## Enforcement

Maintainers follow the [Contributor Covenant Enforcement Guidelines](https://www.contributor-covenant.org/version/2/1/code_of_conduct/#enforcement-guidelines): correction, warning, temporary ban, permanent ban — proportional to the severity and pattern of the conduct.

## Scope

This Code of Conduct applies in all Plinth project spaces and when an individual is officially representing the project in public spaces.
